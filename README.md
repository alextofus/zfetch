# zfetch

### INSTALL
clone repo

cd zfetch

chmod +x zfetch

sudo mv zfetch /usr/local/bin/

### [INSTALL DONE]
now test your fetch by doing
zfetch
------------------------------

### [Reinstall]
1. Remove The zfetch folder
2. Copy This Repo
3. cd zfetch
4. chmod +x zfetch [so you won't get an error]
5. now move the file to /usr/local/bin
